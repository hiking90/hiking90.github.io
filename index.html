Moru (모루)

The anvil where software is forged.

About

In Korean, 모루 (moru) means anvil — the solid, unyielding block on which a blacksmith hammers raw metal into finely crafted tools.

Just as an anvil provides the steady foundation for shaping steel, Moru aims to be the foundation for software developers — providing robust, reliable building blocks forged in Rust so you can focus on crafting what matters.
